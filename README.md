

# Desafio: Conta Bancária Virtual

![image](https://github.com/user-attachments/assets/9157cae0-7abf-4a97-b56b-7c1a00b6374e)

## Descrição
Este projeto tem como objetivo criar uma aplicação simples de controle de conta bancária virtual utilizando Java. A aplicação permitirá ao usuário realizar operações como consultar saldo, receber valores e transferir valores.

## Funcionalidades
- Inicialização dos dados do cliente
- Menu de opções para operações
- Consultar saldo
- Receber valor
- Transferir valor
- Encerrar a aplicação

## Tratamento de Erros
- Se o usuário informar uma opção inválida, a mensagem `Opção inválida` será exibida.
- Se o usuário tentar transferir um valor maior que o saldo disponível, a mensagem `Não há saldo suficiente para fazer essa transferência.` será exibida.
